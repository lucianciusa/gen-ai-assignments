Skillset Enrichment Workflow

A skillset enriches raw content before indexing. The wizard may generate built-in skills for text extraction, chunking, language hints, and embeddings preparation. This enrichment improves downstream retrieval quality.

Typical Steps

  1. Extract raw text from source documents.
  2. Split long content into manageable chunks.
  3. Apply optional normalization or language processing.
  4. Generate embeddings for vector fields.
  5. Map enriched output into index fields.

Why It Matters

  • Chunking avoids losing relevance in very long documents.
  • Structured enrichment improves semantic and vector matching.
  • Consistent transformation supports stable ranking behavior.

Troubleshooting

  • If no text appears in the index, check extraction and field mappings.
  • If vector queries fail, verify vector field type and dimensions.
  • If results are noisy, review chunk overlap and content quality.
