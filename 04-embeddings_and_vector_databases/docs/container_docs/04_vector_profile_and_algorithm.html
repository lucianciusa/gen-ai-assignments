Vector Profile and Algorithm Configuration

The vector profile links embedding generation and nearest-neighbor search behavior. It combines the vectorizer definition with algorithm settings so the search service can retrieve similar chunks efficiently.

Vectorizer Considerations

  • Use a single embeddings deployment for both indexing and query time.
  • Ensure model dimensions match the target vector field.
  • Document the deployment name for reproducible setup.

Algorithm Considerations

  • Use a nearest-neighbor algorithm optimized for high-dimensional vectors.
  • Validate top-k relevance on realistic prompts, not synthetic-only queries.
  • Start with defaults, then tune if quality or latency is insufficient.

Operational Tips

  • Keep chunk size stable during experiments.
  • Track quality changes after each configuration update.
  • Record algorithm and vectorizer settings in your notebook evidence.
