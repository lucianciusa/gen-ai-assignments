Azure AI Search Quick FAQ

Which embeddings model is a good low-cost default?

Use a small embeddings model first, then upgrade only if relevance metrics require it.

Should I choose vector-only or hybrid search?

Hybrid is usually a safer default because it combines semantic similarity and exact matching.

How many documents do I need to test?

Start with at least 8 to 15 focused documents so query behavior can be evaluated across multiple topics.

Why are my semantic results weak?

Check semantic field mappings, title quality, and whether document chunks contain complete ideas.

What should I include in assignment evidence?

Index schema, semantic configuration, vector profile, skillset, indexer status, and query result examples.
