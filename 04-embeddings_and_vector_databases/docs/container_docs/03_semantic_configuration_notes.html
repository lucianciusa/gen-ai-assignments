Semantic Configuration Notes

Semantic ranking improves the ordering of search results by understanding intent and context, not just keyword overlap. In Azure AI Search, semantic configuration defines which fields represent title, body content, and keywords.

Field Mapping Best Practices

  • Title field should capture the section topic clearly.
  • Content field should contain complete sentences and explanatory text.
  • Keyword fields should include domain terms users are likely to query.

Example Query Types

  • "How do I tune vector search for better relevance?"
  • "What is the difference between semantic search and hybrid search?"
  • "Which fields should be filterable in an enterprise index?"

Validation Strategy

  1. Run 10 representative user questions.
  2. Compare baseline keyword ranking against semantic ranking.
  3. Check if top-3 results provide actionable and accurate guidance.
  4. Adjust field mapping if irrelevant results dominate.
